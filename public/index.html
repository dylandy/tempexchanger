How To Deploy

The programme can only deploy and use under Unix-like system (sorry windows) due to the gems (or libraries) used in the project. The following requirement is essential for this project.

Requirements

This project was developed and tested under the following elements. If you can deploy this project other than these elements, please let me know.

  • Unix-like system
  • Ruby
  • Sinatra
  • Nokogiri

After you install all the dependencies and you can download project here and launch the service for temperature exchange service.

How To Use

This api service support four different temperature formats to switch from each other. The current supported formats are "celsius", "fahrenheit", "kelvin" and "reaumur". The return format could be with three different kinds in "plain text", "json" and "xml". Please follow the rule below to use get the data.

  • http://.../[:temperture-format]-to-[:temperture-format]/[:degree]/[:output-format]

Explanation

temperture-formatkelvin, fahrenheit, celsius, reaumur
degreefloat or integer number
output-formatnull, json, xml

Notice:If you want to output plain text, please just leave the output format blank.
